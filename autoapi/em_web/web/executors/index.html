em_web.web.executors.submit_executor(name, *args)[source]¶

Task Executor.

Redis is used to track current executors. When an executor is launched it is added to Redis[“executors”].

The webapp can query /executor/status to get a listing of running tasks.

Parameters

name – name of function to call
