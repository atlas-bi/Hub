:py:mod:`em_web.seed`
=====================

.. py:module:: em_web.seed

.. autoapi-nested-parse::

   Extension script to seed EM2's database.

   Database must be seeded on first run of EM2.

   .. code-block:: console

       export FLASK_APP=em_web
       flask db init
       flask db migrate
       flask db upgrade
       flask seed

       # and for demo
       flask seed_demo
